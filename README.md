# Google Search Ranking – Client

Vite + React + Tailwind app for the Local Search Ranking tool.

## Setup

```bash
cd google-search-ranking/client
npm install
npm run dev
```

Runs at `http://localhost:3000`. Ensure the server is running on port 5524 so `/api` requests work.

## Scripts

- `npm run dev` – dev server
- `npm run build` – production build
- `npm run preview` – preview production build

## Theme

Primary: `linear-gradient(135deg, #ee4679, #a21040)`. White for cards and content. Tailwind breakpoints: default (mobile), `sm` 640px, `md` 768px, `lg` 1024px.
